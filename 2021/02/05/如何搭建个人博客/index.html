如何搭建个人博客

安装hexo

在目录下hexo init

hexo generate / ** 生成一套静态网页 ** /

hexo server / ** 在服务器上运行 ** /

Create a new post

1 $ hexo new "My New Post"

Run server

1 $ hexo server

**More info: ** Server

Generate static files

1 $ hexo generate

**More info: ** Generating

Deploy to remote sites

1 $ hexo deploy
